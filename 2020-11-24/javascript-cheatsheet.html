JavaScript Cheatsheet

Two main kinds of “JavaScript” environments:

1. The runtimes:

  • Browser JavaScript engines - V8 (JavaScript engine) in Chromium (web browser), SquirrelFish Extreme in WebKit and Safari, SpiderMonkey (JavaScript engine) in Mozilla Firefox.
  • Server-side JavaScript engines - V8 in Node.js, and some lesser-known ones.
  • Embedded engines - Rhino (JavaScript) used for embedding in programsJava (programming language).

2. The libraries:

  • Generic JavaScript libraries that do not depend on browsers - Underscore.js.
  • Browser libraries and frameworks - jQuery, BackboneJS, and proprietary ones like rum.js.
  • npm (package manager) modules for Node.js, which can also contain native machine code.
  • CommonJS modules that may be used both with Node.js and in browsers.

Popular JavaScript Engines/Frameworks/Tools

npm: is a package manager for the JavaScript programming language.
React: is an open-source, front end, JavaScript library for building user interfaces or UI components.
Express.js: is a back end web application framework for Node.js, released as free and open-source software (it is designed for building web applications and APIs).
Vue.js: is an open-source model–view–viewmodel front end JavaScript framework for building user interfaces and single-page applications.
Deno: is a runtime for JavaScript and TypeScript that is based on the V8 JavaScript engine and the Rust programming language.
Angular: is a TypeScript-based open-source web application framework led by the Angular Team at Google and by a community of individuals and corporations.
Electron: is an open-source software framework developed and maintained by GitHub.
jQuery: is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animation, and Ajax.
Yarn: is a new package manager that replaces the existing workflow for the npm client or other package managers while remaining compatible with the npm registry.
webpack: is an open-source JavaScript module bundler. It is made primarily for JavaScript, but it can transform front-end assets such as HTML, CSS, and images if the corresponding loaders are included.
AngularJS: is a JavaScript-based open-source front-end web framework mainly maintained by Google and by a community of individuals and corporations to address many of the challenges encountered in developing single-page applications.

Common Vulnerabilities

Prototype pollution: polluting the prototype of a base object which can sometimes lead to arbitrary code execution, prototype pollution happens at some unsafe merge, clone, extend and path assignment operations on malicious JSON objects.

JavaScript is often described as a prototype-based language. The proto property of Object.prototype exposes the internal Prototype (either an object or null) of the object through which it is accessed. The use of proto is controversial, but modern browsers support it anyway.

A common undesired practice is overwriting the prototype of a default JavaScript object. If we modify the prototype of an object in one place, then the behavior of all objects might be affected throughout the entire application.
