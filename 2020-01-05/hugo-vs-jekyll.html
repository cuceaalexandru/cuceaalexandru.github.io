Hugo vs Jekyll

Hello World!

For long I’ve been trying to take some free time and create a blog where I can discuss/save different articles about penetration testing and software development. I will also use this blog as a motivation to try and learn new things.

At first I wanted to create a simple blog in HTML by hand but it required too much editing time for each post/article, so I decided to try Hugo but it was really hard to keep in mind every file used by Hugo themes. I’ve noticed that a certain Hugo theme I liked “hugo-tale” was imported from Jekyll, the next step as you probably have already guessed was to try Jekyll (I’m not a big user of Ruby) but I felt Jekyll was really easy to use and understand.
