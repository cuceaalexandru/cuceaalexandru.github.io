Mobile Security Guide

My security testing guide for Mobile Pentesting with Android or iOS techniques and tools used while doing such engagements.

Android

Tools:

  • JD-GUI
  • Android Studio
  • apktool
  • dex2jar
  • adb

Directory where apps are installed by default: /data/data SD card directory: /sdcard

Tapjacking

General mitigation of Tapjacking: filterTouchesWhenObscured = “true”

More advanced mitigation available in Android API 29:

  • Check the FLAG_WINDOW_IS_PARTIALLY_OBSCURED flag
